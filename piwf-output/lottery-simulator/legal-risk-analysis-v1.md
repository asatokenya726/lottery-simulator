## 2. 法務リスク一覧

### 2-1. 富くじ罪（刑法187条）

| 項目 | 分析 |
|------|------|
| **リスク** | 刑法187条は「富くじの発売」「富くじの取次」「富くじの授受」を禁止。宝くじの確率を再現したシミュレーターが「富くじ」に該当するかが論点 |
| **該当性** | **非該当と判断**。富くじ罪の構成要件は「偶然の事情により当選者を決定し、当選者に財産的利益を与えること」。本サービスは (1) ゲーム内通貨のみ使用し現実の財産的利益を与えない、(2) 換金機能が存在しない、(3) 参加に現実の金銭を必要としない。よって「富くじ」の構成要件を満たさない |
| **対策** | 以下を明示的に表示・記載する: |
| | - 利用規約に「本サービスはシミュレーションであり、実際の宝くじとは無関係です」を明記 |
| | - 「ゲーム内通貨は現実の通貨との交換はできません」を明記 |
| | - 画面常時に「これはシミュレーションです」を表示（M-08） |
| **深刻度** | 低（対策実施後） |

### 2-2. 賭博罪（刑法185条）

| 項目 | 分析 |
|------|------|
| **リスク** | 賭博罪は「偶然の勝負に関し財物を賭ける行為」を処罰。本サービスで賭博が成立するか |
| **該当性** | **非該当**。ユーザーは現実の金銭・財物を一切賭けていない。ゲーム内通貨「ぃえん」は換金不可であり、「財物を賭ける」要件を満たさない |
| **対策** | 換金不可の明示（富くじ罪対策と共通） |
| **深刻度** | 低（対策実施後） |

### 2-3. 景品表示法

| 項目 | 分析 |
|------|------|
| **リスク** | 景品表示法は「一般消費者の利益を保護する」法律。不当な景品類の提供や優良誤認表示を規制 |
| **該当性** | **MVP段階では非該当**。(1) 課金がないため「取引」が発生しない、(2) 景品類の提供がない、(3) 商品・サービスの品質に関する誤認表示の対象となる「商品」がない |
| **将来リスク** | フリマ機能（C-03）導入時にRMT（リアルマネートレード）が発生すると抵触の可能性。法務相談必須 |
| **深刻度** | MVP: なし / 将来: 高 |

### 2-4. 資金決済法

| 項目 | 分析 |
|------|------|
| **リスク** | ゲーム内通貨が「前払式支払手段」に該当する場合、届出義務や供託義務が発生 |
| **該当性** | **非該当**。前払式支払手段の要件は「金額等の財産的価値が記録された証票等で、対価を得て発行されるもの」。ゲーム内通貨「ぃえん」は (1) 無料で付与される、(2) 購入できない、(3) 換金できない。「対価を得て発行」の要件を満たさない |
| **将来リスク** | 課金機能導入時は再検討が必要 |
| **深刻度** | MVP: なし / 将来: 中 |

### 2-5. 著作権・商標

| 項目 | 分析 |
|------|------|
| **リスク** | 宝くじ公式のデザイン・ロゴ・名称の無断使用による著作権・商標権侵害 |
| **該当性** | 注意が必要。「宝くじ」自体は一般名詞であり商標登録されていないが、以下に注意: |
| | - 宝くじの公式ロゴ・デザインの模倣は避ける（C-03制約） |
| | - 「年末ジャンボ」は宝くじの商品名。確率データの引用は事実の記述であり著作権侵害には該当しないが、公式と誤認させるデザインは避ける |
| | - オリジナルのUI/デザインを使用し、公式サイトの模倣をしない |
| **対策** | (1) 全てオリジナルデザインで実装、(2) 「当サービスは公式の宝くじとは無関係です」を免責事項に明記、(3) 確率データの出典を明記（「2024年末ジャンボ公式データに基づく」） |
| **深刻度** | 低（対策実施後） |

### 2-6. ギャンブル依存症対策

| 項目 | 分析 |
|------|------|
| **リスク** | シミュレーションであっても、ガチャのUI/演出がギャンブル依存を助長する可能性があり、社会的批判のリスク |
| **該当性** | 法的義務はないが、社会的責任として対策が望ましい |
| **対策（M-08で実装）** | |
| | - 「これはシミュレーションです。実際のお金は使用されません」の常時表示 |
| | - ギャンブル依存症相談窓口のリンク: 消費者ホットライン（188）、精神保健福祉センター |
| | - 初回アクセス時に注意事項モーダルを表示 |
| **深刻度** | 中（法的リスクは低いが、社会的批判リスクあり） |

## 3. 利用規約で定めるべき事項

以下の項目を利用規約に含めること（M-09で実装）:

| # | 項目 | 必須理由 |
|---|------|---------|
| 1 | **サービスの性質** | 「本サービスはエンターテインメント目的のシミュレーションであり、実際の宝くじの購入・当選とは一切関係ありません」 |
| 2 | **ゲーム内通貨** | 「ゲーム内通貨『ぃえん』は本サービス内でのみ使用できる仮想の通貨であり、現実の通貨・電子マネー・その他の財産的価値との交換はできません」 |
| 3 | **確率データの出典** | 「本サービスで使用する当選確率は、公表されている宝くじの当選確率に基づいていますが、公式の宝くじとは無関係です」 |
| 4 | **免責事項** | 「データの消失、ブラウザの不具合、その他本サービスの利用に起因する損害について、一切の責任を負いません」 |
| 5 | **知的財産権** | 「本サービスのUI/デザイン/コードの著作権は開発者に帰属します」 |
| 6 | **禁止事項** | 「本サービスを利用した営利活動、データの改ざん・不正利用を禁止します」 |
| 7 | **規約の変更** | 「本規約は予告なく変更する場合があります」 |
| 8 | **準拠法** | 「本規約は日本法に準拠します」 |

## 4. プライバシーポリシー

| 項目 | 方針 |
|------|------|
| **必要性** | MVP段階ではプライバシーポリシーの掲載は**推奨だが必須ではない**。理由: 個人情報を一切収集しないため |
| **MVP対応** | 利用規約内に「本サービスは個人情報を収集しません。ゲームデータはお使いのブラウザにのみ保存されます」の1文を含める |
| **将来** | アクセス解析（C-07）導入時はプライバシーポリシーを別ページとして作成する必要がある（Cookie使用の告知、データ収集範囲の明示） |

## 5. 通報・コンテンツモデレーション

| 項目 | 方針 |
|------|------|
| **必要性** | **不要**（MVP段階）。ユーザー生成コンテンツが存在しない（テキスト入力・画像アップロード・コメント機能なし）ため、モデレーションの対象がない |
| **将来** | フリマ機能（C-03）やランキング（C-04）導入時に不正利用の通報機能が必要になる可能性あり |

## 6. 外部サービス利用に伴う法的義務

| 外部サービス | 法的義務 | 対応 |
|-------------|---------|------|
| Vercel（ホスティング） | 利用規約の遵守。無料枠の利用制限に注意 | Vercel ToSを遵守。帯域制限等を確認 |
| GitHub（ソースコード管理） | 特になし（プライベートリポジトリ） | - |
| X/Twitter（SNSシェア） | Twitter API利用規約の遵守（Intent URL利用のため軽微） | Web Intent URL（`https://twitter.com/intent/tweet?text=...`）を使用。APIキー不要 |
| Next.js / npm パッケージ | 各ライセンスの遵守（MIT等） | package.jsonのライセンス確認 |

**越境データ移転**: 本サービスはユーザーデータをサーバーに送信しないため、個人データの越境移転は発生しない。Vercelの CDN でページが配信されるが、個人データを含まない静的コンテンツのため GDPR/個人情報保護法の越境移転規制の対象外。
